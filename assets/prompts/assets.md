# Nano Banana System Prompts: Follow The Bill Website Assets

## Global Design Requirements for All Assets

**CRITICAL DESIGN CONSTRAINTS:**
- **Color Palette**: ONLY pure black (#000000) and pure white (#FFFFFF)
- **No gradients, no grays, no colors** - strictly monochromatic
- **Style**: Minimalist, brutalist, high-contrast
- **Format**: SVG for scalability (unless otherwise specified)
- **Typography**: Serif for logos/headers, sans-serif for body elements
- **Aesthetic**: Clean, geometric, professional, sharp edges

## General Notes for All Assets:

1. **Consistency**: All assets should feel like part of the same family
2. **Scalability**: Prefer SVG for logos, icons, and illustrations
3. **Accessibility**: Ensure high contrast ratios are maintained
4. **Professional Tone**: Avoid anything too playful or casual
5. **Brutalist Aesthetic**: Sharp edges, geometric shapes, bold contrast
6. **No Compromise on Colors**: Absolutely no grays, colors, or gradients

When generating assets, always export in multiple formats where applicable:
- SVG (for web)
- PNG (high-res, with transparency)
- PDF (for print materials)
